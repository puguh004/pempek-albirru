WEBSITE PEMPEK ALBIRRU

Cara membuka:
1. Ekstrak file ZIP.
2. Buka file index.html menggunakan browser.

Cara memasang gratis di GitHub Pages:
1. Buat repository baru di GitHub.
2. Upload seluruh isi folder ini (index.html, styles.css, app.js, dan folder assets).
3. Masuk Settings > Pages.
4. Pada Source pilih Deploy from a branch, branch main, folder /root, lalu Save.
